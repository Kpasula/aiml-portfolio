01 · Professional Bio

Learning in public, building for impact.

I'm a software engineer who spends my days building production web applications and cloud infrastructure — React and TypeScript front ends, Java / Spring Boot services, event-driven systems on Apache Kafka, and AWS infrastructure-as-code. I care about the unglamorous things that make software trustworthy: clean data at the boundary, tests that try to break the code, and interfaces real people can actually use.

Through AIML-500: Machine Learning Fundamentals at Indiana Wesleyan University, I'm extending that engineering mindset into artificial intelligence and machine learning. My goal is simple: to bridge solid software practice with modern AI so the systems I help ship are not just impressive in a demo, but dependable in production.

This portfolio is designed as a resource site, not a filing cabinet. Each artifact shows my thinking and process so a future teammate, employer, or fellow learner can see how I approach a problem — and reuse what's useful.
